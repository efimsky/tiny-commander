## What is Tiny Commander?

Tiny Commander (tnc) is a lightweight dual-pane file manager for Linux and macOS. It's inspired by Midnight Commander but focuses on simplicity and zero external dependencies.

**Key features:**
- Dual-pane interface for efficient file management
- MC-style keybindings
- Integrated command line
- Zero dependencies beyond Python 3.13+ standard library
