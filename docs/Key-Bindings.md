# Key Bindings

Tiny Commander uses Midnight Commander-style keybindings.

## Navigation

| Key | Action |
|-----|--------|
| `↑` | Move cursor up |
| `↓` | Move cursor down |
| `Enter` | Enter directory / Execute command line |
| `Tab` | Switch between left and right panels |
| `Home` | Jump to first file |
| `End` | Jump to last file |
| `Page Up` | Scroll up one page |
| `Page Down` | Scroll down one page |

## File Operations

| Key | Action |
|-----|--------|
| `F3` | View file with pager |
| `F4` | Edit file with editor |
| `F5` | Copy selected files to other panel |
| `F6` | Move selected files to other panel |
| `F7` | Create new directory |
| `F8` | Delete selected files (with confirmation) |

## Selection

| Key | Action |
|-----|--------|
| `Insert` | Toggle selection on current file |
| `+` | Select files by pattern |
| `-` | Deselect files by pattern |
| `*` | Invert selection (toggle all) |

## Menu and Search

| Key | Action |
|-----|--------|
| `F9` | Open/close menu bar |
| `/` | Quick search in current panel |
| `Esc` | Cancel current operation / Close menu |

## Command Line

| Key | Action |
|-----|--------|
| `Alt+Enter` | Insert current filename into command line |
| `Enter` | Execute command (when command line has focus) |

## Advanced Operations

| Key | Action |
|-----|--------|
| `Shift+F3` | Cycle sort order (name → size → date → extension) |
| `Shift+F4` | Create new file (opens in editor after creation) |
| `Alt+F3` | Calculate directory size (cached for session) |

## General

| Key | Action |
|-----|--------|
| `F10` | Quit |
| `Esc` (twice) | Clear command line (when text is present) |

## Menu Bar Navigation

When the menu bar is open (F9):

| Key | Action |
|-----|--------|
| `←` `→` | Switch between menus |
| `↑` `↓` | Navigate menu items |
| `Enter` | Select menu item |
| `Esc` | Close menu |
